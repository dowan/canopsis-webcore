Description

Calendar widget for Canopsis

Screenshots

Installation

You need to clone the git repository and copy directory to Canopsis path

$ git clone https://git.canopsis.net/canopsis-ui-bricks/brick-calendar.git
$ cp -r brick-calendar $CANOPSIS_PATH/var/www/canopsis

Then, you need to enable the brick

$ su - canopsis
$ webmodulemanager enable brick-calendar

You can see enabled bricks

$ su - canopsis
$ webmodulemanager list
[u'core', u'uibase', u'monitoring', ..., **u'brick-calendar'**]
